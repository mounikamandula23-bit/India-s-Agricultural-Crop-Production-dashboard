# ⚡ Visualization tool for India's Agricultural Production Analysis

---

# 📌 Project Overview

The Visualization Tool for India's Agricultural Crop Production Analysis is an interactive data visualization project developed to analyze agricultural production trends across India from 1997 to 2021. The project provides comprehensive insights into crop production, cultivated area, crop yield, seasonal patterns, state-wise and district-wise performance, and overall agricultural productivity.

The project integrates multiple agricultural datasets using MySQL for data management and SQL for data analysis. Interactive dashboards and Tableau Stories are created using Tableau Desktop, published through Tableau Public, and embedded into a Flask web application to provide an intuitive, user-friendly platform for exploring agricultural trends, comparing crop performance, and supporting data-driven decision-making.

# 🎯 Project Objectives

The main objectives of this project are:

- Analyze India's agricultural crop production trends from 1997–2021.
- Compare state-wise and district-wise crop production performance.
- Visualize cultivated area, crop production, and average crop yield.
- Analyze seasonal cultivation patterns and production variations.
- Identify the top-performing crops and states based on production.
- Monitor year-wise agricultural growth and productivity trends.
- Provide interactive dashboards and Tableau Stories for data-driven decision-making in agricultural planning and resource management.
